# Pasta de Imagens dos Projetos

Esta pasta é destinada para armazenar as fotos de capa dos projetos.

## Arquivos necessários:

1. **jogadoras-capa.jpg** - Foto de capa do projeto "Gerenciador de Jogadoras de Futebol Feminino"
2. **ecommerce-capa.jpg** - Foto de capa do projeto "E-commerce Sustentável (EcoTrend)"

## Como adicionar as imagens:

1. Coloque suas imagens nesta pasta (`public/assets/`)
2. Nomeie os arquivos exatamente como:
   - `jogadoras-capa.jpg` (ou .png, .webp, etc.)
   - `ecommerce-capa.jpg` (ou .png, .webp, etc.)
3. O sistema automaticamente usará essas imagens quando disponíveis
4. Se a imagem não for encontrada, será usado um fallback automático

## Formatos suportados:

- JPG / JPEG
- PNG
- WebP
- GIF

## Dimensões recomendadas:

- Largura: 800px - 1200px
- Altura: proporcional (idealmente 16:9 ou 4:3)
- Tamanho do arquivo: otimizar para web (< 500KB recomendado)
